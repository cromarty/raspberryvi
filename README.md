## Sub-directories

* common

Contains code and utilities which are 
not distribution specific.

* alarmpi

Contains code and utilities specific to 
Arch Linux on the Raspberry Pi.

* raspbian

Contains code specific to Raspbian on 
the Raspberry Pi.
